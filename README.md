# Haseeb Arshad - AI Engineer Portfolio

A modern, responsive portfolio website showcasing expertise in artificial intelligence, machine learning, and software engineering. Built with **React**, **Vite**, and **Tailwind CSS**, this portfolio highlights cutting-edge projects and a professional journey in the AI/ML landscape.

## 🚀 Features

### ✨ Interactive & Visual

* 🎇 **3D Particle Background** – Immersive visuals with **Three.js**
* 💫 **Smooth Animations** – Powered by **Framer Motion**
* 📱 **Fully Responsive** – Seamless on **mobile, tablet, and desktop**
* 🌙 **Dark Theme** – Sleek dark mode UI with gradient highlights

### 📂 Portfolio Sections

* **Hero Section** – Dynamic rotating titles & CTA buttons
* **Experience** – Highlights professional milestones
* **Skills** – Core AI, ML, and software technologies
* **Projects** – Case studies & deployed app demos
* **Why Me** – Your unique value proposition
* **Contact** – One-click reach via social platforms

### 🧠 AI/ML-Focused Projects

* Real-world AI use cases
* Model visualizations and explainability
* Cloud-hosted ML deployments

## 📦 Installation & Setup

### 🔧 Prerequisites

* Node.js ≥ 16
* npm or yarn

### 🚀 Quick Start

```bash
git clone https://github.com/Hasee10/portfolio.git
cd portfolio
npm install
npm run dev
```

Visit: [http://localhost:5173](http://localhost:5173)

### 🏗️ Build for Production

```bash
npm run build
npm run preview
```

## 🎨 Customization Guide

| Section      | File                    | What to Edit                            |

## 🌍 Deployment

### 📦 Vercel (Recommended)

1. Push the project to a GitHub repository.
2. Go to [Vercel](https://vercel.com)
3. Click **“New Project”**, select the repo.
4. Set **build command**: `npm run build`
5. Set **output directory**: `dist`
6. Click **Deploy**

✅ Done! Your portfolio is now live.
